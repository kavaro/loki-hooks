createHooksLoki

  • createHooksLoki(dbRegistry: Hooks, collectionRegistry: Hooks): THooksLoki
    • Defined in src/lib/createLoki.ts:25

    Enhances Loki constructor and addCollection method with a hooks option and adds a loaded method to Loki and Collection. The hooks option allows the user to decorate the database and collection methods hooks as enabled by the member-hooks module. The loaded methods will be called after the database or collection has been loaded and can be used in hooks to intercept a load. The createHooksLoki function takes a database and a collection hooks registry (see member-hooks for documentation) as argument and returns the HooksLoki class. This database registry allows the user to define hooks for the database. This collection registry allows the user to define hooks for the collections. The createHooksLoki function should be called only once in the lifetime of the application because it modifies Loki prototype methods
